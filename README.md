1. 字符串
2. 数字类型 Number
3. 列表[],可以储存任何数据类型，可修改
4. 元组(),可以储存任何数据类型，但不能修改

6. 字典{键(Key):值(Value)},键不能重复,和json相似
7. 常见的转义字符
